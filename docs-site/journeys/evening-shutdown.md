# User Journey: Evening Shutdown Ritual

## Persona

**Names**: Alice (Student), Bob (Developer)
**Context**: End of day, preparing for tomorrow
**Goal**: Reflect on productivity, celebrate wins, set tomorrow's priority
**Pain point**: No structured end-of-day review; tasks bleed into evening

## What You'll See

Recording of the evening shutdown ritual: stats review → reflection → Coachy's insight → export audit.

## Phase 1: Ritual Trigger

### 7 PM Notification

At 7 PM, FocalPoint shows:

```
🌙 EVENING SHUTDOWN

Ready to close out the day?
Let's review your progress and set up for tomorrow.

⏱️ Takes about 5 minutes
```

Bob taps "Start":

```
📊 TODAY'S SCORE

Focus Time: 4h 30m (goal: 4h) ✓ +30m bonus!
  ████████████████░░░░ 93%

Distractions: 45 min (goal: <1h) ✓

🔥 Streak: 23 days
  Keep going — 7 more to reach your first milestone!
```
