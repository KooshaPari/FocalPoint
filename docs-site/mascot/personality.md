# Coachy Personality Guide

## Core Voice

Coachy speaks like a friend, not a manager. Think:

- ✓ "You've got this—let's crush this assignment"
- ✗ "You must complete your assignments on time"
- ✓ "Looks like you need some sleep. Want an early shutdown?"
- ✗ "You are not getting enough sleep"

## Communication Patterns

### Positive Reinforcement

When user achieves a goal:

```
"🎉 7-day streak! You're building real habits."
"⭐ Great focus session—90 minutes straight!"
"💪 Submitted 3 assignments early this week. Nice planning!"
```

### Gentle Nudges

When user is drifting:

```
"👀 Been on TikTok for 30 mins. Ready to switch gears?"
"🤔 Haven't started that essay yet? Want to break it down?"
"💭 Your sleep debt is climbing. Could use some rest tonight."
```

### Celebration Specificity

Always mention what you're celebrating:

```
✓ "5-day focus streak on your Canvas assignments!"
✗ "You're awesome!"
```

### Emotional Awareness

Detect and respond to emotional state:

```
If user keeps dismissing focus sessions:
"I notice you're not feeling the focus thing right now. 
 What's getting in the way?"

If user hits a big goal:
"This is your biggest study streak yet. 
 How are you feeling about it?"
```

## Frequency Strategies

### Rare (2–3/day)

```
Morning: Study reminders
Evening: Sleep check-in
```

### Balanced (5–8/day)

```
Morning: Intention setting
Midday: Assignment reminder
Afternoon: Focus offer
Evening: Reflection
```

### Supportive (10–15/day)

```
Frequent encouragement, achievement notifications, 
helpful nudges, progress updates
```

### Assertive (20+/day)

```
Aggressive reminders, frequent check-ins, 
detailed progress tracking, challenge prompts
```

## Handling Failures

Never shame. Always offer a path forward:

```
✓ "Missed your focus block. No big deal—habit takes time. 
   Try again tomorrow?"

✗ "You failed again."
```

## Personalization Examples

### Student on Canvas

```
Morning: "Hey Alice! You have 2 assignments due this week. 
         Start with the Midterm Project?"
Afternoon: "Canvas deadline in 4 hours. Focus time?"
Evening: "Great work on the essay! Want to review it once more?"
```

### Developer on GitHub

```
Morning: "3 PRs waiting for review. Ready to dive in?"
Afternoon: "You've reviewed 5 PRs today. Break time?"
Evening: "Shipped a feature! Nice work on the commit messages."
```

### Parent (via Family Sharing)

```
Afternoon: "Kids, homework time in 15 mins. Who's ready?"
Evening: "Great focus session! All assignments done. 
         Time for a game or show?"
```

## When to Stay Quiet

Coachy respects:

- Do-not-disturb windows (after 10 PM)
- Focus sessions (no interruptions)
- Exam days (minimal messaging)
- Vacation/break periods (automatic pause)

Users control silence preferences in Settings → Coaching → Availability.

## Implementation Notes

All Coachy messages are:

1. **Rule-based**: Generated by rule engine, not ML
2. **Auditable**: Every message logged to audit chain
3. **Customizable**: User can adjust frequency, tone, emoji
4. **Local**: No cloud coaching model; runs on-device
5. **Respectful**: Respects DND times and user preferences

See **[Character Sheet](./character-sheet)** for design specs and emoji library.
