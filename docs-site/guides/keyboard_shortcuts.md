# Keyboard Shortcuts

FocalPoint supports keyboard shortcuts on iOS with external keyboards, iPad, and Mac (Designed for iPad). This guide covers all available shortcuts.

## Global Shortcuts

These shortcuts are available everywhere in the app:

| Shortcut | Action | Context |
|----------|--------|---------|
| ⌘N | New Task | Create a new task from anywhere |
| ⌘F | Start Focus Session | Begin a focus session immediately |
| ⌘. | Cancel Focus Session | Stop the current focus session |
| ⌘1 | Switch to Today | Jump to the Today tab |
| ⌘2 | Switch to Focus | Jump to the Focus tab |
| ⌘3 | Switch to Tasks | Jump to the Tasks tab |
| ⌘4 | Switch to Home | Jump to the Home tab |
| ⌘5 | Switch to Rules | Jump to the Rules tab |
| ⌘6 | Switch to Rewards | Jump to the Rewards tab |
| ⌘7 | Switch to Stats | Jump to the Stats tab |
| ⌘8 | Switch to Coachy | Jump to the Coachy tab |
| ⌘9 | Switch to Activity | Jump to the Activity tab |
| ⌘, | Open Settings | Open the Settings view |
| ⌘⇧S | Sync Now | Manually trigger a sync with connected services |
| ⌘⇧R | Run Rules Now | Manually evaluate and run all active rules |

## Focus Tab Shortcuts

These shortcuts are available when the Focus tab is active:

| Shortcut | Action |
|----------|--------|
| Space | Start/Pause Focus | Toggle focus session start or pause |

## Using Keyboard Shortcuts

### On iPad

Connect a physical keyboard or use the on-screen keyboard with modifiers (⌘). Shortcuts work the same as on Mac.

### On iPhone

Connect a physical keyboard using Bluetooth or a USB adapter. Shortcuts are fully supported with external keyboards.

### On Mac (Designed for iPad)

Use standard Mac keyboard shortcuts. Menu commands are also available in the Mac menu bar:

- **File Menu**: New Task (⌘N)
- **Edit Menu**: Sync Now (⌘⇧S), Run Rules Now (⌘⇧R)
- **View Menu**: Go to Tab (⌘1–⌘9)
- **Focus Menu**: Start Focus Session (⌘F), Cancel Focus Session (⌘.), Keyboard Shortcuts (⌘?)
- **App Menu**: Settings (⌘,)

## Finding Shortcuts in the App

Open **Settings → Support → Keyboard Shortcuts** to view a complete reference of all available shortcuts.

## Tips

- **Tab Navigation**: Use ⌘1–⌘9 to jump between tabs quickly without touching the screen.
- **Focus Control**: Use ⌘F to start a session and Space to pause/resume while in the Focus tab.
- **Batch Operations**: Use ⌘⇧R and ⌘⇧S together to run rules and sync in quick succession.

## Customization

Keyboard shortcuts are currently fixed to the defaults listed above. Future versions may allow customization through Settings.
