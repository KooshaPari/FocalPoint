# Code of Conduct

FocalPoint adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) verbatim.

Full text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## TL;DR

- Be respectful. Assume good faith.
- Critique ideas, not people.
- No harassment, discrimination, or personal attacks.
- Report incidents privately to the maintainer (see `SECURITY.md` for the contact address).

## Scope

This Code of Conduct applies in all project spaces — the repo, issue tracker, discussions, pull requests, docs-site, and any communication channels representing FocalPoint — and when an individual is officially representing the project in public.

## Enforcement

Reports are handled by the project maintainer. Responses range from a private warning to a permanent ban, proportional to the incident. The full enforcement ladder is the one documented in the Contributor Covenant v2.1.
