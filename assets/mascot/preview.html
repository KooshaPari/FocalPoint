Coachy — FocalPoint Parametric Mascot

5 expressions × 4 intensities = 20 deterministic states

Rendering Contract

  • expression: idle, happy, focused, concerned, sleeping
  • intensity: calm, active, intense, post-rule
  • breathPhase: [0–1] continuous oscillation (3000ms default)
  • blinkOpen: boolean for eye state
  • size: SVG viewport width (default 256px)
  • animated: boolean for CSS @keyframes breathing

Idle States

Default baseline. Relaxed, waiting. Used when no rule is active.

Happy States

Positive reinforcement. Smile, squinting eyes. Reward or bypass earned.

Focused States

Concentration. Inward brows, thin mouth. Active rule enforcement.

Concerned States

Warning or low energy. Wavy mouth, worried brows. Auth fail, low battery.

Sleeping States

Resting or dormant. Closed eyes. Rule disabled, quiet hours active.

Calm Intensity

Baseline, relaxed operation. Primary coral color, no badge.

Active Intensity

Engaged and aware. Blue dot badge. Timer running, rule monitoring.

Intense Intensity

High stakes, urgent. Red triangle badge. Block active, penalty pending.

Post-Rule Intensity

Just rewarded. Green body, + badge. Bypass granted, rule paused.
