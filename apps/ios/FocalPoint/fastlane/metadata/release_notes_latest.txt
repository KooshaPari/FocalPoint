FocalPoint v0.2.0 — TestFlight Beta

This is the first TestFlight release of FocalPoint, a dual-mode screen-time companion with parental control and self-control capabilities.

What's included in this beta:
- Connector ecosystem: Canvas, Google Calendar, GitHub integrations for task/calendar sync
- Mascot character system: Coachy guides you through focus blocks and rule enforcement
- Rule evaluation engine: Create and manage focus schedules, app limits, and penalties
- Audit chain: Cryptographically verified log of all rule changes and enforcements
- Wallet system: Earn and spend reward credits for completing focus goals
- iOS background sync: Rules stay active even when the app is closed
- Xcode 26.3+ compatible; tested on iPhone 15 Pro

Known limitations (v0.2.0):
- FamilyControls entitlement approval pending from Apple (blocks enforcement of parental rules)
- Android app deferred to Phase 2
- Web dashboard not yet available
- Multi-device sync deferred to Phase 2
