# FocalPoint v0.0.8 Release Notes

**Release Date:** 2026-04-25  
**RC Soak Completed:** ✅ No regressions. Release-ready for production distribution.

## Overview

FocalPoint v0.0.8 marks the completion of RC.1 soak testing and quality validation. The Helios family (Eidolon, Sidekick, Observably, Stashly, Paginary) is verified green across all platform targets.

## Key Achievements

### Quality & Testing
- **100% FR Traceability**: All 141 v0.0.7 functional requirements now have explicit test vectors (up from 73%)
- **Test Suite**: 180+ assertions passing across 56+ crates
- **Compilation**: Zero warnings, all workspace crates compile cleanly
- **Dead Code Cleanup**: Removed 4 unused functions; suppression count reduced from 69 → 65

### Test Isolation & Reliability
- Fixed environment variable leakage in gcal connector watch tests via serialization locks
- Consolidated duplicate test modules in focus-mascot and focus-coaching
- Enhanced test setup/teardown correctness for cross-test contamination prevention

### Platform Verification
- ✅ iOS platform layer (FFI + CoreHolder)
- ✅ Android platform layer (JNI bindings)
- ✅ Release tooling (release-cut + bench-guard)
