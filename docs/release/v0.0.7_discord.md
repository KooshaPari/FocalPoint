# FocalPoint v0.0.7 — Discord Release Announcement

## Quick Embed Copy

```
✨ **FocalPoint v0.0.7** is ready!

**Headline:** Org-audit integration + template collections + phenotype-tooling lift

🎯 **What's shipped:**
• Org-audit framework (phenotype-bus, dedup, admin sync)
• 7 signed template collections (marketplace + community submissions)
• 9 phenotype-tooling utilities (release-cut, agent-orchestrator, bench-guard, +6)
• Plugin-SDK Phase 2 (HTTP/SSE/WebSocket, bearer auth, rate-limit)
• GraphQL gateway (queries/mutations/subscriptions + auth)
• 141 commits since v0.0.6

🚀 **Platform updates:**
• iOS: MCP bridge, CloudKit Phase 1, lock-screen widgets, Dynamic Island, 6 Siri Shortcuts
• Android: 3 Compose screens (Today/Tasks/FocusTimer)
• Web: Astro marketing site, live preview pane, ReactFlow builder
• macOS: Menubar commands, keyboard shortcuts

📊 **Developer experience:**
• 8 new focus-lang DSL macros (reward/penalize/remind/celebrate/block/unlock_after/track_streak/if_pattern)
• focus-replay time-travel debugger for ruleset exploration
• VSCode extension (FPL syntax + snippets + compile/preview)
• OpenTelemetry tracing + Prometheus metrics integrated

🔗 **Release notes:** https://github.com/KooshaPari/FocalPoint/releases/tag/v0.0.7

Questions? Start here: https://github.com/KooshaPari/FocalPoint#quick-start
```
