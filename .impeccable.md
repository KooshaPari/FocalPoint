# Impeccable — FocalPoint design context

Persistent design guidelines. Consulted by the `frontend-design`, `polish`, `normalize`, `critique`, `clarify`, and related skills.

## Product personality

**Coachy**, the mascot, sets the voice.

- **Terse, warm, honest.** Shorter is better. Never condescending.
- **Coaches, doesn't nag.** "You set this rule. I'm doing my job." — not "naughty you, back on task!"
- **Specific over abstract.** "Canvas says `PSYC 101 essay` is due in 47 min. Instagram is locked until you submit." — not "Stay focused!"
- **Never gamified for gamification's sake.** Streaks and bypass budgets tie to real rules, not confetti.
- **Never uses emoji** unless the user's system actively prefers it. Plain text, strong typography.
- **Acknowledges failure gracefully.** "The Canvas token expired. Re-auth takes 30 s. This rule is paused until you do." — not silent skip.

Reference touchstones: Duolingo's owl (warmth + guilt-trip-light), Oura's evening summary (specific, calm), early Things 3 microcopy (terse, respectful).

## Typography

- **System sans** for prose: `-apple-system, "SF Pro Text", system-ui, sans-serif`.
- **Monospace** for rule DSL, event payloads, audit hashes, CLI output: `"SF Mono", "JetBrains Mono", ui-monospace, monospace`.
- **Scale (modular, base 16):** 12, 14, 16, 18, 22, 28, 36, 48.
- **Weights:** 400 body, 500 UI labels, 600 headings. Never 700 except for hero.
- **Line-height:** 1.5 prose, 1.3 headings, 1.2 dense UI.
- **Tracking:** -0.01em on large headings; 0 elsewhere.

## Spacing & layout

- **Base unit:** 4 px. All spacing is a multiple of 4.
- **Touch target:** minimum 44×44 pt (iOS HIG).
- **Content max-width:** 680 px prose; 960 px mixed; 1200 px dashboards.
- **Grid:** 8-column mobile, 12-column tablet/desktop, 16-px gutter.

## Motion

- **Default easing:** `cubic-bezier(0.2, 0, 0, 1)` (iOS "ease-out-expo").
- **Durations:** 150 ms (micro), 220 ms (standard), 400 ms (enter/exit), 600 ms+ (narrative, Coachy mascot transitions only).
- **Reduced motion:** honor `prefers-reduced-motion`. Coachy still reacts; animations collapse to fades.

## Components

- **Buttons:** rectangular with 8 px corner radius; primary = `--color-focal` fill, text `--color-ink`; secondary = outline in `--color-text-2`.
- **Cards:** `--color-surface` with 12 px radius, 1 px `--color-line` border, no drop shadow by default.
- **Toggles:** iOS-style, 32-pt track height. Use `--color-focal` for "on".
- **Tags / pills:** monospace 12 pt, 4 px vertical padding, 8 px horizontal, 4 px radius, semantic color background at 12 % opacity + full-opacity text.
- **Audit log rows:** monospace hash prefix (first 8 chars), `--color-text-3`; event type bolded; timestamp right-aligned.

## Copy voice — examples

| Bad | Good |
|-----|------|
| "Great job! You're on a 7 day streak! 🔥🔥🔥" | "7-day streak. Bypass budget +1 hour." |
| "Oops! You tried to open Instagram but it's blocked." | "Instagram is locked. Rule `canvas-assignment-due` — 42 min remaining." |
| "Please connect Canvas to continue." | "Canvas is disconnected. Reconnect to resume `assignment-driven-focus`." |
| "Your sleep was not great last night." | "5h 20m sleep last night. Strict-mode enabled until 14:00." |
| "Error" | "Couldn't reach Canvas (HTTP 503). Retrying in 2 minutes." |
